# 🌿 Paradise Nursery

A simple React-based plant shopping app.

## 🚀 Features
- View different plants
- Add plants to cart
- Increase / decrease quantity
- Remove items from cart
- View total price

## 🛒 Technologies Used
- React.js
- JavaScript
- HTML & CSS

## ▶️ How to Run

In the project directory, run:

npm install
npm start

Then open:
http://localhost:3000

## 📦 Build

npm run build
